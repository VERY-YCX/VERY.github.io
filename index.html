第一次为页面开美颜之CSS

    <!DOCTYPE html>
<html>
    <head>
        <title> my page!</title>
    //两种方法用来应用css
    //first 引入 style 标签
        <style>
            body{
                background-color: rgb(168, 162, 162);
                font-family: Arial, Helvetica, sans-serif;
                color:white;

            }如果只设置一款可能用户界面上不能呈现
            h1{
                color:aquamarine;
            }
            h2{
                color:red;
                opacity: 0.6;
            }
            button
            {
                background-color: blanchedalmond;
                border: none;
                border-radius: 30px;
            }
            //border:none 使button边框消失，*border_radius:用来添加圆角边框
        </style>

    //second 引入 link 标签,link 标签允许我们加载外部样式表，从而实现一个样式表被多个页面共用
        
          <link rel="stylesheet" href="">
    </head>
    <body>
        <h1>My webpage!</h1>
        <h2>About me!</h2>
        <p>
            I am a handsome guy,
            eighteen years old now.<br>
            I am learning <em>software</em> in <strong>UESTC</strong>
        </p>
            <button>Click to learn more!</button>
    </body>
</html>

//css 属性：color ，opacity(透明度) ，background-color, font(字体)

初遇html

    <!DOCTYPE html>
<html>
      <head>
            <title>My page!</title>
      </head>

      <body>
        <h1>First tltle</h1>//告诉浏览器知道页面中的重要内容,大小不是直接区别而层级关系
                                才是，可以通过css来改变字体大小
        <h2>Second tiltle</h2>
        <h6>The min title</h6>    
        <p>My passage text </p>//文本标签 ，不同段落默认css样式带换行
                               //html 除了一个该有的空格外，不关心多余的空格，空格只是为了更好地阅读  
        <p>hi my name is <br>kevin!</p>//br 用来换行
        <p><em>yes</em>i'm in <strong>uestc</strong></p>//strong 用来进行字体加粗
                                                       //em 用来强调 会使字体变斜
    <p><button>I am a button!</button></p> //暂时未用js来定义按钮的跳转行为

//列表类标签 :
//ul:unorder list; 
ol: order list;
li: list item;
a: anchor(锚，铁锚的)；
i：italic斜体（文本）；
b: bold粗体（文本）；

        <ul>
            <li>i am a item</li> 
            <li>i am a item </li>//会出现小黑点
        </ul>
        <ol>
            <li>i am a item</li>
            <li>i am a item</li>//会出现数字序号
        </ol>
</body>
</html>
