title:有关css选择器的介绍

基础选择器是如何工作的 ：
空格意味着左边内部的所有右边
逗号两边都设置指定样式

精确设置css：

 <section class="feature-box sales" > 
                        <div>section a</div>
                        <div>section b</div>
                        <div>section c</div>
 </section>
 <section class="feature-box closeouts">
                        <div>section d</div>
                        <div>section e</div>
 </section>



section.feature-box{
            display: flex;
        }

section.feature-box.sales div{
                background:gray;
                width:10px;
                margin: 1px;
            }

修改后：

section.feature-box{
                display: flex;
            }
            .feature-box div{
                width: 10px;
                margin: 1px;
            }
            section.feature-box.sales div{
                background: grey;
            }
            section.feature-box.closeouts div{
                background:red;
            }
