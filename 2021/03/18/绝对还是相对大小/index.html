title：简单的布局

//编写网页首先要学会如何对页面进行布局

//div标签 （divider）div实际上就是一个独立占一行，从左到右横跨内容的盒子
多个div会垂直排列,基本每个网页的小区快都用div表示

//语义化的标签 方便阅读理解也可以帮助搜索引擎识别代码
常用的：header nav（代表导航栏，导航栏中的每一项用ul+li） section footor

//理解结构化布局的含义

<!DOCTYPE html>
<html>
    <head>
    <title>my page!</title>
    </head>
    <body>
        <h1>Let's build a layout!</h1>
        <div>a</div> //div*4可以快捷地打出
        <div>b</div>
        <div>c</div>
        <div>d</div>
    </body>
</html>

//主要学会在不同元素或标签之间增加间距
在css中有两种方法来增加间距 margin and padding

//margin:20px 表示在上下左右都增加了20px的间距
//如果是两个值那么第一个代表上下边，第二个值代表左右边
//如果是四个值那么则上右下左顺时针方向

//在css中有多个单位如20%（占浏览器窗口比例）， 20px

 <!DOCTYPE html>
<html>
    <head>
        <title>my page!</title>
        <style>
            body{
                background-color: grey;
                color :white;
                font-family: Arial, Helvetica, sans-serif; 
                margin: 0;
                padding: 0;
            }

            //默认情况下<body>自带内外边距，不同浏览器的默认值不同

            header{
                background: yellow;;
                margin:20px;    
            }                   
                                
            section{
                background: violet;
                padding:20px;
            }
            footer{
                background: turquoise;
                pading:10px,20px;
            }
        </style>
    </head>
    <body>
        <header> header</header>
        <section> triple-section</section>
        <section>lower-section</section>
        <footer>footer</footer>    
    </body>
</html>
